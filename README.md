# SangToProject
### (only Korean) 웹크롤링을 이용한 한국어 끝말잇기

※본 프로젝트는 학교 활동의 일환으로 진행된 프로젝트입니다. 학교 이름 등의 개인정보 유출 확인 시 즉시 수정할 예정이오니 양해부탁드립니다.

#### 2022.11.25
